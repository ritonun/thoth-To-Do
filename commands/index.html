CLI Reference

This page provides documentation for our command line tools.

thoth

Usage:

thoth [OPTIONS] COMMAND [ARGS]...

Options:

  --help  Show this message and exit.

a

Add a new todo

Usage:

thoth a [OPTIONS] TODO_STRING

Options:

  --help  Show this message and exit.

init

Init the application

Usage:

thoth init [OPTIONS]

Options:

  --help  Show this message and exit.

ls

List all todo

Usage:

thoth ls [OPTIONS]

Options:

  --help  Show this message and exit.

rm

Remove a todo

Usage:

thoth rm [OPTIONS]

Options:

  --help  Show this message and exit.

s

Search for a todo

Usage:

thoth s [OPTIONS]

Options:

  --help  Show this message and exit.
